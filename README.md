# anycase

> Convert strings to any case

## Usage

```nim
import anycase

camel("any case")   # "anyCase"
kebab("any case")   # "any-case"
pascal("any case")  # "AnyCase"
path("any case")    # "any/case"
snake("any case")   # "any_case"
plain("any-case")   # "any case"
```

## Supported cases

- `camel`
- `kebab`
- `pascal`
- `path`
- `snake`
- `plain`
